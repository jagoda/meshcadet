DejaVu Sans — license notice
=============================

DejaVuSans.ttf is NOT bundled in this repository (confirmed by `git ls-tree
-r HEAD`: no DejaVu file is tracked). It is read, at firmware build time
only, from a system-installed path —
`/usr/share/fonts/truetype/dejavu/DejaVuSans.ttf` (the `fonts-dejavu-core`
package on Debian/Ubuntu; see `firmware/build.rs`) — by
`firmware/gen_emoji_font.c`, which rasterizes a curated subset of its ASCII
glyphs into the `MeshCadetEmoji` bitmap font baked into the firmware binary
(alongside glyphs derived from NotoEmoji-Regular.ttf; see
`firmware/assets/NotoEmoji-LICENSE.txt`).

That rasterized output is a Modified Version of DejaVu Sans under the terms
below (Bitstream Vera Fonts Copyright, which DejaVu is built on and licensed
under, plus DejaVu's own public-domain changes), so this notice — including
the copyright statement — travels with this repository per that license's
own redistribution condition, even though no DejaVu font file itself is
checked into the tree. See `NOTICE` at the repository root and
`docs/licensing/DEPENDENCY-AUDIT.md` for the full attribution/audit context.

Fonts are (c) Bitstream (see below). DejaVu changes are in public domain.
