Mission Control // Web Flasher

Flash MeshCadet

Install MeshCadet firmware on a LilyGo T-Deck Plus straight from your browser, over a USB cable — no toolchain, no command line.

1. Connect your T-Deck Plus

Plug the T-Deck Plus into this computer with a USB-C cable. Use a desktop build of Google Chrome or Microsoft Edge — this page needs Web Serial, which Firefox, Safari, and mobile browsers don't support.

2. Choose a version

3. Choose how to flash

Fresh install writes the whole firmware image (bootloader, partition table, and app) and erases everything on the device. Upgrade writes only the new app firmware in place and leaves your device's identity, config, and conversation history untouched — but only when your device is already running a build with a compatible partition layout (see ADR-0009).

4. Install

Flashing erases and reinstalls the whole firmware image (bootloader, partition table, and app) — any settings or contacts stored on the device are cleared, the same as a fresh build.

Your browser doesn't support installing firmware over USB. Use a desktop build of Chrome or Edge. This page needs to be loaded over HTTPS to install firmware (it is here — if you're seeing this, something's off; please open an issue).

4. Back up, then install

Upgrade is designed to preserve your conversation history, but flashing over USB can still fail partway — export a transcript first if that history matters to you. Open the provisioner's history export →

Your browser doesn't support installing firmware over USB, or this page isn't loaded over HTTPS. Use a desktop build of Chrome or Edge, loaded over HTTPS (or from localhost).

Prefer to flash manually, or want an older/newer version than what's listed? Grab the merged image (manifest.json) or the app-only update image (manifest-update.json + update-meta.json) straight from the GitHub Releases page instead.
